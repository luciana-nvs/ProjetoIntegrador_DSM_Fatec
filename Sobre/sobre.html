Sobre

Este projeto acadêmico do curso de Desenvolvimento de Software da Fatec Diadema visa criar um sistema web simples usando HTML, CSS e JavaScript para auxiliar consumidores a encontrar perfumes alinhados às suas preferências olfativas e a fornecer insights básicos para desenvolvedores de fragrâncias.

Ele foi concebido em resposta a uma necessidade concreta e atual do mercado, em estreita colaboração com o curso de Cosméticos da mesma instituição.

Com o auxílio de questionários e ferramentas interativas, o sistema identifica e define o perfil olfativo do usuário. Isso permite que eles encontrem perfumes que se alinhem perfeitamente às suas preferências individuais, tornando a escolha de fragrâncias mais personalizada e satisfatória.

Vale ressaltar que este projeto está em uma fase de desenvolvimento contínuo e, embora ainda não tenha sido finalizado, representa uma proposta futura. Isso possibilitará aos consumidores a maravilhosa jornada de descobrir fragrâncias verdadeiramente personalizadas, capazes de proporcionar uma experiência olfativa singular e enriquecedora.
